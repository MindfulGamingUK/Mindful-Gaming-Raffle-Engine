# Prize Image Sourcing Guide

For the production raffle engine, legitimate product images must be used.

## Recommended Sources
1. **Official Brand Assets**: Use Press/Media kits from brands (e.g., Sony, Microsoft, Apple).
2. **Retailer Feeds**: Partner with retailers or use high-quality placeholder URLs from official listings.
3. **Internal Photography**: For unique or used prizes, take custom photos to ensure authenticity and build trust.

## Image Specifications
- **Hero Image**: 1600px width, aspect ratio 16:9 or 4:3.
- **Gallery Images**: Min 800px, consistent aspect ratio.
- **Alt Text**: Descriptive for accessibility and SEO (e.g., "Sony PS5 Pro Console with dual controllers").

## License Tracking
- Use the `imageSourceNote` field in the raffle data to track licenses (e.g., "Promotional use only - Sony Press Kit").
- Avoid using "placeholder.com" in production; it looks unprofessional and reduces trust.
